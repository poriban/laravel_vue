# laravel vue

## 概要
- ドライバー配送アプリ ライドシェアの解禁のニュースを見て少しプロダクトを考えてみようと思い作りました。
- laravelはあくまでAPIのみ
- twilioからトークン発行

- 目的地までの道のりを表示
- 完了まで
